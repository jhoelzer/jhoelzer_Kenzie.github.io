HTML Notes

HTML: Hypertext Markup Language

HTML is a markup language that tells browsers how to structure a web page. It consists of a series of elements, which alter the way the page looks and behaves.

Block-level elements form a visible block on a web page, appearing on a separate line from the content before and after it. Inline elements are contained within block-level elements and only surround small parts of the content. These normally appear inside a paragraph text and will not cause a new line to appear in the document. Empty elements are elements that consist only of a single tag (no closing tag), such as the "img" tag.

Attributes are additional parts of a tag that contain information about the element. An attribute should have:

  • a space between it and the element name
  • the attribute name followed by an equals sign
  • an attribute value, with opening and closing quotation marks around it
